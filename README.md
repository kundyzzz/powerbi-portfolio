# RetailBloom UK - Sales Analytics Dashboard

Power BI дашборд для анализа продаж розничного магазина.

## Страницы:
- Sales Overview — общая выручка, заказы, клиенты
- Products Analysis — топ продуктов по выручке
- Customers Analysis — анализ клиентов по географии
- Product Detail — детализация транзакций

## Инструменты:
Power BI, DAX
